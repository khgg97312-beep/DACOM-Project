#  DACOM - 해커톤 협업 플랫폼

## 💡 서비스 소개
DACOM은 해커톤 참가자들이 팀을 구성하고 소통하며 프로젝트를 효율적으로 수행할 수 있도록 지원하는 협업 플랫폼입니다.

## 🎯 주요 기능
- 팀 매칭 기능
- 채팅 기반 소통 기능
- 프로젝트 관리 기능

## 🖥️ 실행 방법
pip install -r requirements.txt  
streamlit run app.py

## 🔗 Demo
